01

KOTLIN BASICS

Kotlin is a statically typed programming language that runs on the Java Virtual Machine and can be compiled to JavaScript source code.

fun main() {
    // Print to console
    println("Hello, Kotlin!")
    
    // Variables
    val name = "CyberDev" // Immutable
    var count = 0 // Mutable
    count += 1
}
02

SYNTAX & DATA TYPES

Kotlin provides a concise syntax with type inference and null safety built into the type system.

// Data types
val number: Int = 42
val decimal: Double = 3.14
val text: String = "Kotlin"
val isReady: Boolean = true
val list: List<String> = listOf("a", "b", "c")

// Null safety
var nullableString: String? = null
val length = nullableString?.length ?: 0
03

FUNCTIONS

Kotlin functions are declared using the 'fun' keyword. They support default arguments, named parameters, and extension functions.

// Basic function
fun greet(name: String): String {
    return "Hello, $name"
}

// Single-expression function
fun square(x: Int) = x * x

// Function with default parameter
fun connect(
    host: String,
    port: Int = 8080
) { /* ... */ }

// Extension function
fun String.reverse() = this.reversed()
04

OBJECT-ORIENTED PROGRAMMING

Kotlin supports OOP with classes, objects, interfaces, and data classes that reduce boilerplate code.

// Class with primary constructor
class User(val name: String, var age: Int)

// Data class (automatically implements equals, hashCode, toString)
data class Person(
    val name: String,
    val email: String
)

// Interface
interface Repository {
    fun getById(id: Int): Any?
    fun save(item: Any)
}
05

KEY FEATURES

Kotlin includes many modern language features that make development more productive and enjoyable.

  • Null Safety - Eliminates NullPointerExceptions
  • Extension Functions - Add functionality to existing classes
  • Smart Casts - Automatically casts types after checks
  • Data Classes - Reduce boilerplate for POJOs
  • Coroutines - Asynchronous programming made easy
  • Type Inference - No need to explicitly declare types
  • Interoperability - Seamless Java integration
06

CONTROL FLOW

Kotlin provides familiar control flow structures with some enhancements for better expressiveness.

// If expression (returns a value)
val max = if (a > b) a else b

// When expression (enhanced switch)
val description = when (x) {
    0 -> "zero"
    1 -> "one"
    in 2..10 -> "small number"
    else -> "large number"
}

// For loop
for (i in 1..5) { println(i) }
for (item in collection) { println(item) }
