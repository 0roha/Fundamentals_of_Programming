Variables & Data Types

C# is a strongly-typed language, meaning every variable has a specific type.

int age = 25;
string name = "Neo";
bool isConnected = true;
double height = 1.85;
char initial = 'N';

Common data types: int, string, bool, double, char, float, decimal

Control Structures

Control the flow of your program with conditionals and loops.

// If statement
if (temperature > 30) {
  Console.WriteLine("It's hot!");
}

// For loop
for (int i = 0; i < 10; i++) {
  Console.WriteLine(i);
}

Methods

Methods are blocks of code that perform specific tasks.

static int AddNumbers(int a, int b) {
  return a + b;
}

// Calling the method
int result = AddNumbers(5, 3);

Classes & Objects

C# is an object-oriented language where classes define blueprints for objects.

public class Robot {
  public string Name { get; set; }
  public int PowerLevel { get; set; }

  public void Activate() {
    Console.WriteLine($"{Name} is now active!");
  }
}

// Creating an object
Robot r2d2 = new Robot();
r2d2.Name = "R2-D2";
r2d2.Activate();

Arrays & Collections

Store multiple values in a single variable using arrays and collections.

// Array
int[] numbers = new int[] { 1, 2, 3, 4, 5 };

// List
List<string> names = new List<string>();
names.Add("Alice");
names.Add("Bob");
names.Add("Charlie");

Exception Handling

Handle runtime errors gracefully using try-catch blocks.

try {
  int result = 10 / 0; // This will cause an exception
}
catch (DivideByZeroException ex) {
  Console.WriteLine("Error: Cannot divide by zero!");
  Console.WriteLine(ex.Message);
}
finally {
  Console.WriteLine("This code always executes");
}
