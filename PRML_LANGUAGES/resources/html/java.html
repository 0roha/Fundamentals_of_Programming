1. Introduction

Java is a high-level, class-based, object-oriented language that runs on the JVM. It emphasizes portability, security, and a strong standard library.

// Basic Java program
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, Java!");
  }
}

2. Basic Syntax

Programs are organized into classes. Methods contain statements ending with semicolons. Java is case-sensitive.

// Class and main method
class Example {
  public static void main(String[] args) {
    // statements
  }
}

3. Data Types

Primitives: byte short int long float double char boolean

int age = 20;
double price = 19.99;
char letter = 'A';
boolean ok = true;

4. Variables

Local, instance, and static variables. Declare then initialize.

// instance vs static
public class Person {
  String name;          // instance
  static int count;     // static
}

5. Operators

Arithmetic, relational, logical, assignment, and increment/decrement operators.

int x = 5 + 3; // 8
boolean b = (x > 10) && (x < 20);
x++;

6. Control Flow

If/else, switch, for/while/do-while loops.

// if-else
if (x > 10) {
  System.out.println("x > 10");
} else {
  System.out.println("x <= 10");
}

// for loop
for (int i = 0; i < 5; i++) {
  System.out.println(i);
}

7. Arrays

Fixed-size collections of the same type.

int[] nums = {10, 20, 30};
System.out.println(nums[0]); // 10

8. Strings

Immutable sequences of characters. Useful methods: length(), toUpperCase(), substring().

String s = "Java";
System.out.println(s.length());
System.out.println(s.toUpperCase());

9. Methods

Methods (functions) belong to classes; can be static or instance.

public class Calculator {
  static int add(int a, int b) { return a + b; }
  public static void main(String[] args) {
    System.out.println(add(2,3));
  }
}

10. Object-Oriented Programming (OOP)

Four pillars: Encapsulation, Inheritance, Polymorphism, Abstraction.

class Car {
  String brand;
  int year;
  void drive() { System.out.println(brand + " driving"); }
}

11. Inheritance

Subclass extends a superclass and inherits members.

class Animal { void sound(){ System.out.println("Animal sound"); } }
class Dog extends Animal { void sound(){ System.out.println("Dog barks"); } }

12. Polymorphism

Objects can be treated as instances of their parent type. Method overriding is runtime polymorphism.

Animal a = new Dog();
a.sound(); // Dog barks

13. Abstraction

Abstract classes declare methods to be implemented by subclasses.

abstract class Shape { abstract void draw(); }
class Circle extends Shape { void draw(){ System.out.println("Circle"); } }

14. Interfaces

Interfaces declare methods that implementing classes must provide.

interface Animal { void sound(); }
class Dog implements Animal { public void sound(){ System.out.println("Dog"); } }

15. Exception Handling

Use try/catch/finally to handle runtime errors cleanly.

try {
  int r = 10 / 0;
} catch (ArithmeticException e) {
  System.out.println("Error: " + e.getMessage());
} finally {
  System.out.println("Cleanup");
}

16. Packages & Import

Organize classes into packages and import libraries with import.

import java.util.Scanner;
Scanner sc = new Scanner(System.in);

17. Collections Framework

Common interfaces: List, Set, Map. Implementations include ArrayList, HashSet, HashMap.

List list = new ArrayList<>();
list.add("Java");
System.out.println(list);

18. File Handling

Use java.io (or java.nio) to read/write files; handle IOExceptions.

try (FileWriter fw = new FileWriter("out.txt")) {
  fw.write("Hello");
} catch (IOException e) {
  e.printStackTrace();
}

19. Multithreading

Create threads by extending Thread or implementing Runnable. Use synchronization where needed.

class MyThread extends Thread { public void run(){ System.out.println("running"); } }
new MyThread().start();

20. JVM & Memory

The JVM divides memory into stack (locals) and heap (objects). The garbage collector reclaims unused objects.
