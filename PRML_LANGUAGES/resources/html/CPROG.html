INTRODUCTION

C is a general-purpose programming language created by Dennis Ritchie at Bell Labs in 1972. It's one of the most influential programming languages and forms the foundation for many modern systems.

hello.c 
#include <stdio.h>

int main() {
    printf("Hello, Cyber World!\n");
    return 0;
}

C provides low-level access to memory and requires minimal runtime support, making it ideal for system programming like operating systems and embedded systems.

KEY FEATURES

  • Procedural programming language
  • Low-level memory access via pointers
  • Structured programming support
  • Minimal runtime requirements
  • Efficient and fast execution
  • Highly portable across platforms
  • Rich set of operators
  • Static type system
pointers.c 
// Pointer example - Direct memory access
int value = 42;
int *ptr = &value;  // Pointer to value

printf("Value: %d\n", value);
printf("Address: %p\n", ptr);
printf("Value via pointer: %d\n", *ptr);

DATA TYPES

C has several built-in data types that allow programmers to efficiently manage memory.

datatypes.c 
// Basic C data types
char letter = 'A';          // 1 byte
int number = 100;          // 4 bytes
float decimal = 3.14;      // 4 bytes
double precise = 3.14159;  // 8 bytes
short small = 32767;       // 2 bytes
long big = 2147483647;     // 8 bytes

// Derived types
int array[10];             // Array of 10 integers
struct Point {             // Structure
    int x;
    int y;
};

C also supports type modifiers like signed, unsigned, short, and long to customize data types.

CONTROL FLOW

C provides standard control structures for decision making and looping.

control.c 
// If-else statement
if (temperature > 30) {
    printf("It's hot!\n");
} else if (temperature > 20) {
    printf("It's warm.\n");
} else {
    printf("It's cool.\n");
}

// For loop
for (int i = 0; i < 10; i++) {
    printf("Iteration %d\n", i);
}

// While loop
int count = 5;
while (count > 0) {
    printf("Countdown: %d\n", count);
    count--;
}

C also includes switch statements for multi-way branching and do-while loops for post-test loops.

FUNCTIONS

C is a procedural language, so programs are divided into functions that perform specific tasks.

functions.c 
// Function declaration (prototype)
int add(int a, int b);

// Function definition
int add(int a, int b) {
    return a + b;
}

// Recursive function example
int factorial(int n) {
    if (n <= 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

// Main function - program entry point
int main() {
    int result = add(5, 3);
    printf("5 + 3 = %d\n", result);
    
    int fact = factorial(5);
    printf("5! = %d\n", fact);
    
    return 0;
}

CYBER APPLICATIONS

Even after 50+ years, C remains essential in the cyber age for critical systems:

  • Operating Systems: Linux, Windows, macOS kernels
  • Embedded Systems: Microcontrollers, IoT devices
  • Cybersecurity Tools: Network scanners, packet analyzers
  • Game Engines: Performance-critical components
  • Blockchain: Cryptocurrency implementations
  • Databases: MySQL, PostgreSQL core components
  • Compilers: GCC, Clang are written in C
